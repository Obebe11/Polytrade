# 🤖 Polymarket Trading Bot

Автоматический торговый бот для Polymarket — ищет дешёвые контракты (0.001–0.03$),
размещает лимитные ордера близко к рынку, управляет риском и диверсификацией.

---

## 📋 Что умеет бот

| Функция | Описание |
|---|---|
| 🔍 Поиск сделок | Сканирует все активные рынки, фильтрует контракты по цене 0.001–0.03$ |
| 📊 Лимитные ордера | Ставит BUY-ордер чуть выше лучшего аска (fast-fill) |
| 🌍 Диверсификация | Не более 2 позиций в одной категории за цикл |
| ⚠️ Риск-контроль | 2% портфеля на сделку (диапазон 1–3%) |
| 🚫 Исключения | Пропускает рынки выборов (elections/politics) |
| 📅 Срок | Только рынки с 3 днями < expiry < 1 год |
| ⏱ Автоматизация | Сканирование каждый час |

---

## 🚀 Быстрый старт

### 1. Требования
- Python 3.11+
- Polygon-кошелёк с USDC.e и POL (для газа)

### 2. Установка

```bash
# Клонируйте / скачайте файлы бота
cd polymarket_bot

# Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate      # Linux/Mac
# venv\Scripts\activate       # Windows

# Установите зависимости
pip install -r requirements.txt
```

### 3. Настройка

```bash
# Скопируйте шаблон конфига
cp .env.example .env

# Откройте .env и заполните
nano .env
```

Минимально нужно заполнить только **PRIVATE_KEY** и **PORTFOLIO_VALUE**.

### 4. Запуск

```bash
python bot.py
```

Бот запустится, деривирует API-ключи из приватного ключа и начнёт первое сканирование.

## ⚙️ Настройка параметров

В начале `bot.py` есть блок «Параметры стратегии»:

```python
MIN_PRICE          = 0.001   # минимальная цена контракта
MAX_PRICE          = 0.030   # максимальная цена контракта
RISK_PCT           = 0.02    # риск на сделку (2% от портфеля)
MIN_DAYS_TO_EXPIRY = 3       # минимум дней до истечения
MAX_DAYS_TO_EXPIRY = 365     # максимум дней до истечения
SCAN_INTERVAL_SEC  = 3600    # интервал сканирования (1 час)
MAX_POSITIONS_PER_CATEGORY = 2   # макс. позиций в одной категории
PRICE_OFFSET       = 0.001   # насколько выше аска ставить ордер
```

## 🔐 Безопасность

- **Никогда** не публикуйте `.env` с приватным ключом в git
- Добавьте `.env` в `.gitignore`
- Используйте отдельный кошелёк только для бота с небольшой суммой
- Начните с `PORTFOLIO_VALUE=10–50$` чтобы протестировать

## 📊 Как работает алгоритм

```
Каждый час:
  1. Загрузить все активные рынки (Gamma API)
  2. Для каждого рынка:
     а) Получить лучший аск (CLOB API)
     б) Проверить: цена в диапазоне?
     в) Проверить: срок истечения 3–365 дней?
     г) Проверить: не выборы?
     д) Проверить: категория не переполнена?
  3. Рассчитать размер позиции = (портфель × 2%) / цена
  4. Выставить лимитный ордер по цене аск + 0.001$
  5. Сохранить позицию в positions.json
  6. Ждать следующего цикла
```

## 🛠 Решение проблем

**L2_AUTH_NOT_AVAILABLE**: Неверный приватный ключ или тип подписи. Проверьте PRIVATE_KEY.

**insufficient balance**: Пополните кошелёк USDC.e на Polygon.

**Geoblock**: Вы находитесь в ограниченном регионе. Используйте VPN.

**insufficient allowance**: Нужно одобрить расходование токенов через Polymarket UI (одноразово).
